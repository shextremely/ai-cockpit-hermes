## 端口速查（当前环境）

| 服务 | 端口 | 说明 |
| --- | --- | --- |
| Hermes API | **5173** | `%HERMES_HOME%\.env`（本机 `AppData\Local\hermes`）中 `API_SERVER_PORT` |
| 驾驶舱前端(Vite 开发) | **5174** | 与 Hermes 5173 错开 |
| 驾驶舱 BFF | **5180** | 代理 Hermes、托管生产静态资源 |

> 生产模式 `npm run build && npm start` 后，浏览器只需访问 **http://localhost:5180**（同源，无需单独开前端端口）。
